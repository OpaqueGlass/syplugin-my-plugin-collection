## fakeDocBreadcrumb （伪）文档面包屑

[English](https://github.com/OpaqueGlass/syplugin-my-plugin-collection/blob/main/fakeDocBreadcrumb/README_english.md)

> 当前版本：v0.1.0

### 快速开始

- 开启插件即可；

#### 说明

本插件：

- 不支持鼠标悬停时显示浮窗；
- 手机端无法使用；
- 不支持显示同一层级的其他文档，只能向上层跳转；
