## hierarchyNavigate 文档标题下添加上下层文档导航

[English](https://github.com/OpaqueGlass/syplugin-my-plugin-collection/blob/main/hierarchyNavigate/README_english.md)

> 当前版本：v0.1.0

### 快速开始

- 开启插件即可；

#### 说明

和HBuilderX-Light在此功能实现上相比，本插件：

- 在父文档不存在时将展示其同级文档；
- 不支持鼠标悬停预览；
- 在安卓App测试可用（大概）；

## 反馈bug
